## If you need to re-download fonts:

Run the download script from the project root:
```bash
bash fonts/download-fonts.sh
```

Or manually download from Google Fonts CDN using the URLs fetched from:
- Roboto: `https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap`
- Noto Sans Devanagari: `https://fonts.googleapis.com/css2?family=Noto+Sans+Devanagari:wght@400;700&display=swap`

## Important:
- Filenames must match **exactly** as shown above
- Fonts must be valid TrueType (.ttf) files, not HTML or other formats
- All fonts have been tested and can be registered successfully
